Для коректної роботи транскрибації необхідно що б на комп'ютері були встановленні Python 3.9 - 3.11, та Node js 20.16 (Lts версія).
Для встановлення залежностей необхідних для роботи транскрибації запускаємо із правами адміністратора start_install_dependencies.bat.
Для запуску програми транскрибації запускаємо із правами адміністратора start.bat.
Після того як start.bat відпрацював заходимо у браузері http://localhost:5173/ і вносимо налаштування з'єднання із базою даних,
та місце розташування аудіофайлів. Як що з'єднання із базою даних успішне зберігаємо з'єднання (кнопка зберегти) та запускаємо транскрибування
файлів (кнопка старт).
